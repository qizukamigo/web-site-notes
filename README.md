# web-site-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常包含站点相关的笔记、备忘、截图或信息整理，以静态页面形式呈现，便于查阅和分享。

仓库本身不绑定任何特定域名或网站，所收录的页面内容均来自日常浏览与记录，仅作个人存档与参考用途。

## 目录说明

```
web-site-notes/
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
└── index.html      # 仓库首页（可选，用于展示页面列表）
```

- `pages/`：每个 HTML 文件对应一个独立的归档页面，文件名通常反映页面主题或来源标识。
- `assets/`：按需存放页面引用的资源文件，保持目录结构清晰。
- `index.html`：可选的导航页面，便于快速浏览所有归档内容。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开，不依赖外部服务。
- 页面内容以原始信息为主，可能包含少量格式调整或注释，以提升可读性。
- 归档时尽量保留页面原有结构，必要时会添加说明性文字。
- 不保证所有页面内容持续更新或完整准确。

## 维护说明

- 本仓库不定期更新，新增页面会按顺序添加到 `pages/` 目录。
- 若发现归档页面存在明显错误或失效，可提交 Issue 或 Pull Request。
- 仓库维护者保留对内容进行修改或删除的权利。
- 欢迎通过 GitHub 仓库的 Issues 或 Discussions 进行交流。

## 许可

本仓库内容仅供个人学习与参考，不涉及商业用途。如需引用或转载，请注明出处。
